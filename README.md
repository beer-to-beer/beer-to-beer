## Getting Started

**run following commands**

**OS x**
```sh
brew install imagemagick
bundle install
rake db:migrate
rake db:seed
```

**Ubuntu**
```sh
sudo apt-get install imagemagick imagemagick-doc
bundle install
rake db:migrate
rake db:seed
```
